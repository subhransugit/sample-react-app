
# Sample React App (With Auth + Products)

## Features
- Login / Logout (auth protected)
- Dashboard
- Products page: add / delete / view
- Protected routes using localStorage

## Run Instructions

### 1. Install dependencies
```bash
npm install
```

### 2. Run the app
```bash
npm start
```

### 3. Login Credentials
- Username: `demo`
- Password: `secret`

## Tech Used
- React
- React Router
- LocalStorage-based Auth
